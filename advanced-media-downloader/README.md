Advanced Media Downloader - Split Repos
======================================

This archive contains four minimal repositories to run an advanced media downloader system:
- telegram-bot
- download-engine
- fastapi-backend
- react-dashboard
- infra (docker-compose and .env)

Quick start (local, requires Docker):
1. Edit infra/.env
2. cd infra
3. docker-compose up --build -d

Each repo also contains a requirements.txt and simple Dockerfile.
